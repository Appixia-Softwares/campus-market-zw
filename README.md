# 🌟 Campus Market ZW

> **Empowering students to buy, sell, and trade within their campus community.**

Campus Market ZW is a vibrant, student-driven marketplace designed to make campus life easier, more affordable, and more connected. Whether you want to buy textbooks, sell gadgets, find accommodation, or offer services, this is your go-to platform for all things campus!

## 🚀 Get Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Appixia-Softwares/campus-market-zw.git
   cd campus-market-zw
   ```
2. **Install dependencies (with pnpm):**
   ```bash
   pnpm install
   ```
3. **Run the app:**
   ```bash
   pnpm dev
   ```
4. **Build for production:**
   ```bash
   pnpm build && pnpm start
   ```

## 🛠️ Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS, PostCSS, CSS Modules
- **State & Forms:** React, React Hook Form, Zod
- **UI:** Radix UI, Framer Motion, Custom Components
- **Database:** Supabase (Postgres)
- **Auth:** Supabase Auth, Middleware
- **PWA:** Workbox, Service Workers
- **Package Manager:** pnpm

---

## 🧩 Project Structure

- `/app` — Main app pages (marketplace, accommodation, dashboard, help, etc.)
- `/components` — UI components (carousels, chat, navigation, forms, etc.)
- `/hooks` — Custom React hooks
- `/lib` — Utilities, Supabase client, actions
- `/public` — Static assets (logo, images)
- `/styles` — Global and utility styles

## ❓ FAQ

<details>
<summary>How do I create an account?</summary>
To create an account, click on the "Sign Up" button in the top right corner. Provide your email, create a password, and verify your student status.
</details>

<details>
<summary>How do I list an item for sale?</summary>
After signing in, go to the Marketplace and click on "Sell an Item". Fill out the form, add photos, set a price, and submit your listing.
</details>

<details>
<summary>How do payments work?</summary>
Campus Market facilitates in-person exchanges. Meet in a safe, public location on campus and arrange payment directly with the buyer or seller.
</details>

<details>
<summary>How do I verify my student status?</summary>
After creating your account, go to your Profile and click on "Verify Student Status". Upload a photo of your student ID or provide your university email for verification.
</details>

<details>
<summary>What if I have an issue with an order?</summary>
First, try to resolve it directly with the buyer or seller via messaging. If unresolved, contact our support team for help.
</details>
